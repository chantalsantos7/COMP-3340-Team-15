If you have admin access this document will provide help with all the different admin functionalities and hopefully address any additional questions that you may have

Add Product


If you access your user account page you will find a link to a form called Add Product. Select this link and fill out the form accordingly. This is connected to our back end site database and will populate a database record in the Items table that will then be displayed on our main site depending on the category you entered. If you enter incorrect information that does not align with our database no changes will occur.

Remove Product


If you access your user account page you will find a link to a form called Remove Product. Select this link and fill out the form accordingly. This is connected to our back end site database and will remove a database record in the Items table if and only if the name you entered on the form matches with a name in the database. If no match is found no changes will be reflected on the main site

Edit Product


If you access your user account page you will find a link to a form called Edit Product. Select this link and fill out the form accordingly. This is connected to our back end site database and will match with a record in the Items table depending on the values you enter in the form. If you enter incorrect information that does not align with our database no changes will be reflected on the site.

Set Layout Theme


If you access your user account page you will find a link to a page called Set Layout Theme. Select this link and choose the color of the site from 3 different options. The changes you make should be immediately reflected and change the color of the site for every user even after you logout.

List of users and Emails


If you access your user account page you will find a link to a page called List of usernames and emails. Select this link and you should be brought to a page that connects to our database and displays all registered usernames and emails

View all orders


If you access your user account page you will find a link to a page called View all orders. Select this link and you should be brought to a page that connects to our database and displays all previously completed orders that have been made through the site

Cart


If you access your user account page you will find a link to a page called Cart. Select this link and you should be brought to a page that functions the same as our regular user cart page where you can test certain functionalities of the cart
